# Agro Forte - Futuro Sustentável

## Equilíbrio entre Produção e Meio Ambiente

Um site completo e bem elaborado sobre o agronegócio sustentável, apresentando a trajetória do agro brasileiro, os pilares da sustentabilidade, inovações tecnológicas e a visão de futuro para um agronegócio regenerativo e responsável.

### 📋 Conteúdo do Site

O site apresenta as seguintes seções principais:

#### 1. **Trajetória do Agronegócio Brasileiro**
- Histórico desde a década de 1960
- Evolução da modernização agrícola
- Conceito de "Efeito Poupa-Terra"
- Dados sobre mecanização e produtividade

#### 2. **Pilares da Sustentabilidade**
- **Tecnologia:** Agricultura digital, drones, sensores e IA
- **Preservação:** Recuperação de áreas degradadas e sistemas ILPF
- **Energias Limpas:** Biomassa e energia solar
- **Responsabilidade Social:** Agricultura familiar e segurança alimentar

#### 3. **Inovações Tecnológicas**
- Agricultura de Precisão
- Bioinsumos
- Rastreabilidade via Blockchain
- Drones e Sensores
- Inteligência Artificial
- Sistemas Integrados (ILPF)

#### 4. **Galeria Visual**
- Imagens de alta qualidade mostrando práticas sustentáveis
- Tecnologias em ação no campo

#### 5. **Dados e Estatísticas**
- Tabela com indicadores de produtividade
- Crescimento da mecanização agrícola
- Evolução da Produtividade Total dos Fatores (PTF)

#### 6. **Futuro Sustentável**
- Visão de um agronegócio regenerativo e circular
- Compromisso com a resiliência climática

### 🚀 Como Usar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/jokerthur/Agro-forte-futuro-sustent-vel-equil-brio-entre-produ-o-e-meio-ambiente.git
   cd project_repo
   ```

2. **Abra o arquivo index.html em um navegador:**
   ```bash
   # No Linux/Mac
   open index.html
   
   # No Windows
   start index.html
   ```

3. **Ou use um servidor local:**
   ```bash
   # Com Python 3
   python3 -m http.server 8000
   
   # Com Node.js
   npx http-server
   ```

   Depois acesse: `http://localhost:8000`
