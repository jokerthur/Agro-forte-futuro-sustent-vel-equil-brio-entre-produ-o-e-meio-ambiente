# 🌱 Agro Forte - Futuro Sustentável

Um site moderno e responsivo sobre agronegócio sustentável, inovação tecnológica e práticas agrícolas do futuro.

## 🎯 Sobre o Projeto

**Agro Forte** é um movimento transformador que une a força da agricultura tradicional com as inovações tecnológicas mais avançadas. Este site apresenta:

- **Agricultura de Precisão**: Monitoramento em tempo real com drones e sensores IoT
- **Gestão Inteligente de Água**: Economia de até 40% no consumo hídrico
- **Análise de Dados Avançada**: Big data e machine learning para decisões assertivas
- **Sustentabilidade**: Práticas que preservam o meio ambiente
- **Comunidade**: Colaboração entre agricultores, pesquisadores e empresas

## ✨ Características

✅ Design moderno com paleta Modernismo Sustentável

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/jokerthur/Agro-forte-futuro-sustent-vel-equil-brio-entre-produ-o-e-meio-ambiente.git
```

2. Abra o arquivo `index.html` no seu navegador

3. Ou use um servidor local:
```bash
python -m http.server 8000
# Acesse http://localhost:8000
```
